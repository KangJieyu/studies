6.函数	也是一个对象
  创建：
	  构造函数创建对象 var func = new Function("语句;");
	  函数声明创建对象 function func([para]) {
	  					语句;
	  				 }
	  函数表达式创建对象var func = function([para]) {
	  						语句;
	  				  }
  实参，形参：
	  传递时不会检查类型，也不会检查实参数量  val + undefined = NaN
	  实参可以是函数，函数的嵌套调用；
	  定义形参相当于在函数中声明了变量。
  返回值：
	  可以是任何类型	return value;
	  默认不写为undefined
  func()和func的区别：func()调函数，func函数对象
  
  立即执行函数：(function() {
  			  
  			  })();
  			  只会执行一次。
  方法：
  	  call(this,para...)	apply(this,Array)	调用函数，使函数执行
  	  可修改函数中的this指向
  	  
  作用域：
	  全局作用域
	  	在浏览器页面打开时创建，关闭时结束；
	  	全局对象window，浏览器的窗口，由浏览器创建，全局作用域中创建的属性都保存在window对象，创建的函数作为window的方法；
	  	变量的声明提前
	  		用var声明的变量，会在所有的代码执行之前被声明，但不会进行赋值，
	  		没有使用var声明的变量，则不会提前声明。
	  		函数中var声明的变量也声明提前。
	  	函数的声明提前
	  		使用函数声明形式创建的函数，在所有代码执行之前被创建，可以在声明前调用。
	  		函数中嵌套函数也提前。
	  局部作用域
	  	函数执行时创建，完毕时销毁。
	  	函数中不使用var声明的变量都作为全局的。

10.4 事件冒泡bubble
事件向上传导。当子元素事件触发的同时也会触发父类及祖先元素的事件
绑定事件取消冒泡 cancelBubble = true;
	 事件委派
将事件统一绑定给元素的祖先，所以子代元素事件触发时会冒泡到祖先元素

10.5 事件传播
- 捕获阶段
	由外向内，默认不会触发事件
- 目标阶段
	捕获到了元素，不会触发事件
- 冒泡阶段
	事件捕获结束，触发事件
